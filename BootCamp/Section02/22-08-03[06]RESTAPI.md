### REST API

> REST API란 ❓
>
> REST는 "Representational State Transfer"의 약자로 웹(http)의 장점을 최대한 활용할 수 있는 아키텍처이다. REST API는 웹에서 사용되는 데이터나 자원(Resource)을 HTTP URI로 표현하고, HTTP 프로토콜을 통해 요청과 응답을 정의하는 방식을 말한다.

💡 API를 메뉴판이라 생각하면 이해에 도움이 된다.
