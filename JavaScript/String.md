# JS String(문자열)

### 정의 방법

1. 큰 따옴표
```javascript
let myname = "Heropy"
```

2. 작은 따옴표
```javascript
let email = 'test@gmail.com'
```

3. 백틱 with 보간법
```javascript
let hello = `Hello ${myName}'
```
<br>

### 보간법이란

`$`(달러사인)과 중괄호`{ }`를 사용하여 기존에 존재하는 변수값을 `${}` 안에 할당하여 사용하는 방식을 뜻한다.

> 주의❗️
>
> 보간법을 쓰려면 문자열을 반드시 백틱(`)을 사용하여 정의해야 한다.
