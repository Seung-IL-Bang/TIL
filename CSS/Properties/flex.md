# CSS flex

부모 요소에 `display: flex;` 를 적용하면 자식 요소가 수평 정렬된다. **이때, 부모 요소를 Flex Container라고 하고, 자식 요소를 Flex Items 라고 한다.**

<br>

### Flex Container에 사용 가능한 속성들
- display
- flex-flow
- flex-direction
- flex-wrap
- justify-content
- align-content
- align-items

<br>

### Flex Items에 사용 가능한 속성들
- order
- flex
- flex-grow
- flex-shrink
- flex-basis
- align-self
